# profile
This is Masaki Ikeda's profile page. There is also a contact form where you can make inquiries and report problems.

# Requirements

- PHP 8.4.4+
- Composer
- MySQL 
- Laravel 12.0.1+
- Git
- macOS

# installation

```bash
git clone https://github.com/MasakiIkeda0629/lara_final.git

composer install

php artisan migrate
```
# .env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=contact
DB_USERNAME=root
DB_PASSWORD=root
